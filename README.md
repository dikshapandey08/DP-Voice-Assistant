# DP-Voice-Assistant

## Overview

DP (Digital Personal Assistant) is a Python-based desktop voice assistant that understands voice commands and automates everyday computer tasks. It uses SpeechRecognition and pyttsx3 to provide a natural voice-based interaction.

## Features

- Voice Command Recognition
- Text-to-Speech Response
- Open Google
- Open YouTube
- Search Wikipedia
- Tell Current Time
- Send Emails
- Open VS Code

## Tech Stack

- Python
- SpeechRecognition
- pyttsx3
- Wikipedia
- smtplib
- webbrowser
- datetime

## Installation

pip install -r requirements.txt

python jarvis.py
